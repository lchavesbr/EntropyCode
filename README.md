# Description

This project aims to implement some of Entropy Codes in Modern C++, such as Huffman and Arithmetic algorithms.
It will (try to) follow best practices of files/folder structures, clang-format and static analyses (cppcheck + clang-tidy) toolchains integrated in development workflow.
